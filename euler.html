Euler Rotations

In the recapitulation of angular momentum theory, we see that the axis of rotation \(\hat n\) can be described by two parameters, \(\theta \in [0, \pi]\) and \(\varphi \in [0, 2\pi)\), where \(\hat n = \sin\theta(\hat x\cos\varphi + \hat y\sin\varphi) + \hat z\cos\theta\). The rotation around \(\hat n\) is \(\hat R_{\hat n}(\phi) = e^{-i\phi\hat n\cdot\hat{\vec J}/\hbar}\). This is all well and good, but it's inconvenient to have three parameters associated with a single operator, so we turn to Euler rotations (also called Euler angles).

Euler Rotations

Given angles \(\alpha, \beta, \gamma\), Euler rotation \(R(\alpha, \beta, \gamma)\) are performed as follows:
  1. Rotate about the \(\hat z\)-axis by \(\alpha\): \(\hat R_{\hat z}(\alpha)\)
    • \(\hat x \to \hat x' = \hat R_{\hat z}(\alpha)\hat x = \hat x\cos\alpha + \hat y\sin\alpha\)
    • \(\hat y \to \hat y' = \hat R_{\hat z}(\alpha)\hat y = \hat y\cos\alpha - \hat x\sin\alpha\)
    • \(\hat z \to \hat z' = \hat z\)
  2. Rotate about the \(\hat y'\)-axis by \(\beta\): \(\hat R_{\hat y'}(\beta)\)
    • \(\hat x_i^" = \hat R_{\hat y'}(\beta)\hat x_i'\)
  3. Rotate about the \(\hat z^"\)-axis by \(\gamma\): \(\hat R_{\hat z^"}(\gamma)\)
    • \(\hat x_i^{'''} = \hat R_{\hat z^"}(\gamma)\hat x_i^"\)
Thus, \(\hat R(\alpha, \beta, \gamma) = \hat R_{\hat z^"}(\gamma)\hat R_{\hat y'}(\beta)\hat R_{\hat z}(\alpha)\). We have what we want, one parameter per operator. However, each axis of rotation depends on the previous rotations, which is still pretty inconvenient. We can separate them even more by taking advantage of the fact that \(\hat R_{\hat n}(\phi) = \hat R\hat R_{\hat z}(\phi)\hat R^{-1}\), where \(\hat R\hat z = \hat n\). Essentially, any rotation about an axis \(\hat n\) can be performed by first rotating the entire system so that \(\hat n\) is aligned with the \(\hat z\)-axis, performing the rotation about the \(\hat z\)-axis, and then rotation the system again so that \(\hat n\) is oriented how it was originally.

First we expand the rotation \(\hat R_{\hat z^"}(\gamma)\): \[\begin{align} \hat z^" &= \hat R_{\hat y'}(\beta) \hat R_{\hat z}(\alpha) \hat z \\ \hat R_{\hat z^"}(\gamma) &= \hat R_{\hat y'}(\beta) \hat R_{\hat z}(\alpha) \hat R_{\hat z}(\gamma) \hat R_{\hat z}^{-1}(\alpha) \hat R_{\hat y'}^{-1}(\beta) \end{align}\] and then substitute it into \(\hat R(\alpha, \beta, \gamma) = \hat R_{\hat z^"}(\gamma)\hat R_{\hat y'}(\beta)\hat R_{\hat z}(\alpha)\): \[\begin{align} \hat R(\alpha, \beta, \gamma) &= \hat R_{\hat z^"}(\gamma)\hat R_{\hat y'}(\beta)\hat R_{\hat z}(\alpha) \\ &= \hat R_{\hat y'}(\beta) \hat R_{\hat z}(\alpha) \hat R_{\hat z}(\gamma) \hat R_{\hat z}^{-1}(\alpha) \hat R_{\hat y'}^{-1}(\beta) \hat R_{\hat y'}(\beta)\hat R_{\hat z}(\alpha) \\ &= \hat R_{\hat y'}(\beta) \hat R_{\hat z}(\alpha) \hat R_{\hat z}(\gamma) \hat R_{\hat z}^{-1}(\alpha) \hat R_{\hat z}(\alpha) \\ &= \hat R_{\hat y'}(\beta) \hat R_{\hat z}(\alpha) \hat R_{\hat z}(\gamma) \\ \end{align}\] and now we expand the rotation \(\hat R_{\hat y'}(\beta)\): \[\begin{align} \hat y' &= \hat R_{\hat z}(\alpha) \\ \hat R_{\hat y'}(\beta) &= \hat R_{\hat z}(\alpha) \hat R_{\hat y}(\beta) \hat R_{\hat z}^{-1}(\alpha) \end{align}\] and then plug it in: \[\begin{align} \hat R(\alpha, \beta, \gamma) &= \hat R_{\hat y'}(\beta) \hat R_{\hat z}(\alpha) \hat R_{\hat z}(\gamma) \\ &= \hat R_{\hat z}(\alpha) \hat R_{\hat y}(\beta) \hat R_{\hat z}^{-1}(\alpha) \hat R_{\hat z}(\alpha) \hat R_{\hat z}(\gamma) \\ &= \hat R_{\hat z}(\alpha) \hat R_{\hat y}(\beta) \hat R_{\hat z}(\gamma) \\ \end{align}\] Perfect; \(\hat R(\alpha, \beta, \gamma) = \hat R_{\hat z}(\alpha) \hat R_{\hat y}(\beta) \hat R_{\hat z}(\gamma)\) is now three single-parameter rotations whose axes are not dependent on \(\alpha, \beta, \gamma\).
